Configures analog input.

Parameters
SourceChannelChannel to convert (0 to 15)
ConversionTriggerSourceTrigger source for the ADC.
SampleTimeSample time for auto mode (only relevant for Auto mode).
Returns
Nothing

ADCconfig assumes that all PORT B pins may be used for analog inputs. The function turns ADC module off, prior to configuration, and turns the module on again, at the end.

Some configurations are static:

  • All 16 pins (AN0 to AN15) configured to analog input.
  • Format is Integer 16-bit
  • Voltage sources are AVdd and AVss
  • No scanning of multiple inputs
  • Number of conversions per interrupt = 1
  • Buffer fill mode: one 16-word buffer
  • Always use MUX A
  • ADC clock source is PB Clock
  • ADC clock prescaler is set to 7
