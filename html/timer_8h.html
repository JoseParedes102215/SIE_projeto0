◆ TypeBTimer16bitSetFreq()

int TypeBTimer16bitSetFreq ( TypeBTimerNo_t  TimerNo,
uint32_t  Freq_Hz 
)

Sets the counting frequency of a Type B Timer.

If the requested frequency is not possible, the Timer registers are not changed and the function returns NOK. When the requested frequency is possible, the Timerx (x=2..5) registers are set accordingly and the function returns OK.

If some time number is not implemented, TypeBTimerSetFreq returns NOK

Parameters
TimerNoNumber of Timer to configure
Freq_HzRequested frequency (in Hz)
Returns
OK: sucess; NOK: failure
